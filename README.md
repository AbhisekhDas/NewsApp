 ## 🤔 What is this project about?
 In this project I built a news-app based on Reactjs and NewsAPI. The app shows the top 20 headlines of the day.


## 🛠️  Built with

 - [Reachtjs](reactjs.org): Frontend
 - [Axios](https://www.npmjs.com/package/axios): Pulling data from the API
 - [NewsAPI](https://newsapi.org/) : News Data
